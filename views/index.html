Whaaat's the time?

*This server hosts a microservices API

API Usage:

You can return the date using ISO format yyyy-mm-dd

/api/{yyyy}-{mm}-{dd}

You can also return the date from microseconds from zero time.

/api/{microseconds}

API Example:

Call By Date:

/api/2015-12-25

Response:{'timestamp':2015-12-25}

Call by Microseconds:

/api/1451001600000

Response:{'timestamp':2015-12-25}
